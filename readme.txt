Widget radio-radio streaming di Indonesia yang mendukung untuk di semua browser dan juga di SmartPhone.


== Description ==
Kumpulan streaming radio Islam di Indonesia yang Insya Allah berdasarkan pengajaran *Al Qur'an* dan *As Sunnah [Al Hadits]* dan pemahaman para *Shahabat*. Tambahkan *Radio Islam Indonesia* di widget atau dengan shortcode <code>[radio-islam-indonesia]</code>. Semoga bermanfaat.


== Installation ==
Untuk menampilkan *Radio Islam Indonesia* ini, cukup tambahkan widget *Radio Islam Indonesia* di widget dashboard anda. Atau, jika anda ingin menampilkan *Radio Islam Indonesia* di page, post atau di mana saja, cukup menggunakan shortcode *<code>[radio-islam-indonesia]*.


== Frequently Asked Questions ==

= Lokasi untuk menampilkan *Radio Islam Indonesia*? =
Lebih bagus jika anda menampilkan *Radio Islam Indonesia* ini di sidebar, saya sudah menyiapkan widgetnya, sehingga anda cukup drag n drop saja di sidebar yang anda inginkan

= Apakah *Radio Islam Indonesia* mendukung di SmartPhone? =
Insya Allah, karena *Radio Islam Indonesia* menggunakan jPlayer yang sudah compatible dengan semua browser termasuk smartphone browser, baik itu untuk iOS atau Android.

= Bagaimana menambah list radio di *Radio Islam Indonesia*? =
Saya sangat menghargai bagi anda yang sudah mengerti akan programming. Anda sangat mudah merubahnya. Tetapi mohon dengan amat sangat agar anda tidak melakukan hal itu. *Jika anda ingin menambahkan list di Radio Islam Indonesia bisa menghubungi saya di <a href="mailto:darto@kloningspoon.com">darto@kloningspoon.com</a>, sehingga plugin ini bisa terupdate otomatis dan lebih bermanfaat untuk user lainnya yang sudah memasang plugin Radio Islam Indonesia*.

Silahkan dikirim dengan mencantumkan *Nama Radio*, *Frekuensi*, *Lokasi*, *Alamat website*.
